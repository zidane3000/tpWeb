TP Web : Javascript et HTML5
Description
Ce projet est une simple démonstration de la programmation orientée objet en JavaScript, spécifiquement pour créer des formes sur un canvas HTML5.

Structure
Le projet est divisé en plusieurs parties :

Model : Définit les formes de base (Shape, Rectangle, Line, Circle) qui peuvent être créees et avec lesquelles on peut interagir.
View : Définit comment les formes sont dessinées sur le canvas.
Interaction : Gère les événements de la souris pour dessiner les formes sur le canvas.
Controller : Gère la logique de l'application, en coordonnant le modèle et la vue. Il reçoit les entrées de l'utilisateur (par exemple, les clics de souris), met à jour le modèle en conséquence, et met à jour la vue pour refléter les changements dans le modèle.
Installation
Pour installer ce projet, suivez ces étapes :

Clonez le dépôt à partir de GitHub(tp1_V1_readme).
Ouvrez le fichier HTML dans un navigateur web.
Utilisation
Après avoir ouvert le fichier HTML dans un navigateur web, vous pouvez interagir avec le canvas en cliquant et en faisant glisser la souris pour créer des formes, vous pouvez aussi modifier le width du pinceau et la couleur. Une liste des differents shapes dessinés est crée juste a coté pour voir detail ( les dimensions, couleur, ...) et pour supprimer les shapes.
